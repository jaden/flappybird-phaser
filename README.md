# Flappy Bird clone in Phaser

The code is based on this tutorial:

- http://blog.lessmilk.com/how-to-make-flappy-bird-in-html5-1/
- http://blog.lessmilk.com/how-to-make-flappy-bird-in-html5-2/

## Note
Files need to be served with a web server.  File URLs won't work.

## To Play
Browse to index.html and use the spacebar or click to flap your wings.
